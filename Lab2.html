External data

Use the read_excel() function from the readxl package to read in Excel data sets. For example:

library(readxl)
Elevator_Data  = read_excel("ElevatorDataLab1.xlsx")

Please note that if instead you were loading in a CSV file (i.e., Comma-separated Values), you would use the read.csv command. For example:

library(readxl)
Elevator_Data  = read.csv("ElevatorDataLab1.csv")

Note: you put either the relative directory or the absolute directory in both functions. Refer to https://en.wikipedia.org/wiki/Directory_(computing).

Visualizations

Here we use the mtcars dataset in the datasets package. First, we use the plot() command to visualize this data and then we use ggplot().

Hint: When calling data, you can either use data=mtcars or type mtcars$mpg~mtcars$wt. This resource discusses common operators in R.

plot(mpg ~ wt, data = mtcars) # formula notation

# Using ggplot:
library(ggplot2)
ggplot(mtcars, aes(x = wt, y = mpg)) +
  geom_point()

qplot is a shortcut designed to be familiar if you’re used to creating plots using the command plot(). It can create the same plots as ggplot, but ggplot() is easier to create complicated plots.

# Using qplot:
qplot(wt, mpg, data = mtcars)

Basics of ggplot

The ggplot package has seven aspects, but we are only going to focus on the first three:

  • Data
  • Aesthetics
  • Geometries
  • Facets
  • Statistics
  • Coordinates
  • Themes

Think about each aspect as a layer of your plot.

The first layer is your data—which makes sense intuitively, since those are the variables we are going to plot.

The second layer is aesthetics or mapping, which is how you want to plot the data—variables to use. It is specified inside aes() below—anything inside aes() depends on your data.

Finally, you decide what kind of plot you want for your data, such as a scatter plot, histogram, box plot, or bar plot. Geometries are chosen via a series of geom functions, such as geom_point() for scatter plots.

ggplot(data = mtcars, aes(x = hp, y = mpg, size = cyl)) + geom_point(color = 'blue') + ggtitle("Scatterplot")

Exampes

Scatterplot

Scatterplots are best for showing relationship between two numeric variables, as shown above.

Histogram

Histograms are used to display the distribution of numeric variables. It only needs one variable to show. You can adjust the plot with bin or binwidth.

mtcars$cyl = as.character(mtcars$cyl)
ggplot(data = mtcars, aes(x = mpg, fill = cyl)) + geom_histogram(bins = 15) + ggtitle("Histogram")

Dot plots and box plots

Dot plots are useful when you want to show the distribution of either numeric or categorical variables.

ggplot(mpg, aes(x = class, y = displ)) + 
  geom_dotplot(binaxis='y', stackdir='center') + ggtitle("Dot plot")

Box plots are another way of showing distribution of numeric variables, with the 1st and 3rd quartiles represented by the edges of the box.

ggplot(mpg, aes(x = class, y = hwy)) + 
  geom_boxplot() + ggtitle("Box plot")

Bar plot

A bar plot illustrates data in rectangular bars with length of the bar proportional to the value of the variable.

Below, we use dplyr to manipulate the data. You will learn more about this in next week lab, but here is a cheat sheet for how to use this package.

library(dplyr)
species_mean = iris %>% group_by(Species) %>% 
  summarise(Petal.Length.mean  = mean(Petal.Length))

ggplot(data=species_mean, aes(x=Species, y=Petal.Length.mean)) +
  geom_bar(stat="identity") + ggtitle("Mean of Petal Length for Different Species")

Density plot

Density plots are continuous version of histogram. They convey the same information—distribution of numeric variables. Instead of bins, density plots use a kernel density estimate (smooth curve) to represent the distribution. If you compare the density plot with the histogram, you can see they are showing the same thing.

ggplot(data = mtcars, aes(x = mpg, fill = cyl)) + geom_density() + ggtitle("Density plot")
